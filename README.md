# Abhishek Kaniganti — Portfolio

Award-winning portfolio website with stunning animations, dark theme, and glassmorphism design.

## Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, responsive
- **JavaScript** — Vanilla ES6+
- **GSAP + ScrollTrigger** — Scroll & entrance animations
- **tsParticles** — Neural network particle background
- **Lenis** — Smooth momentum scrolling
- **Google Fonts** — Syne + JetBrains Mono + Inter

## Local Preview

Simply open `index.html` in your browser. No build step required.

## GitHub Pages Deployment

1. Create a new GitHub repository
2. Push this folder to the `main` branch:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

3. Go to **Settings → Pages → Source** → select `main` branch → Save
4. Your site will be live at `https://YOUR_USERNAME.github.io/YOUR_REPO/`

## Features

- 🎨 Dark theme with electric purple, teal, and amber accents
- ✨ Neural network particle animation in hero
- 🖱️ Custom animated cursor (dot + ring with lerp lag)
- 📜 Smooth scroll with Lenis momentum
- 🎬 GSAP entrance and scroll-triggered animations
- ⌨️ Typewriter effect cycling through role titles
- 📊 Animated stat counters
- 🃏 3D tilt effect on project cards
- 📱 Fully responsive (320px → 1920px)
- ♿ Respects `prefers-reduced-motion`
